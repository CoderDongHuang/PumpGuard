# Kaggle — Industrial Equipment Monitoring Dataset

---

## 基本信息

| 项目 | 内容 |
|------|------|
| **名称** | Industrial Equipment Monitoring Dataset |
| **上传者** | dnkumars |
| **平台** | Kaggle |
| **设备类型** | 涡轮机、压缩机、**水泵** |
| **特征** | 温度、压力、振动、湿度 + 故障标签（二元） |

## 数据规格

- 多设备类型覆盖 → 可做跨设备迁移学习
- 含 `faulty` 标签 → 可用于故障检测分类
- 含运行环境数据（湿度等） → 关联环境因素

## 本方案用途

- 多设备类型故障检测模型验证
- 环境因素与故障关联分析
- 跨设备迁移学习实验
