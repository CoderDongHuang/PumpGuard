> 飞书 2026 AI 先锋未来人才大赛 — 利欧集团企业命题参赛方案
>
> 让每一台泵都能"说话"：物理机理 + 数据双驱动的泵群预测性维护与健康管理系统。

---

## 项目概览

本项目面向利欧集团提出的工业 AI 命题——如何借助 AI 构建水泵设备的「智慧预测与健康管理」系统，让全球百万台泵从"哑设备"变成会主动报告健康状态、提前预警故障、自动触发运维决策的智能体。

**核心设计理念**：
- **存量兼容、增量智能** — 新泵老泵都能接入，不绑架硬件改造
- **机理兜底、数据进化** — 物理模型保证零故障样本不乱报警，数据模型随维修反馈持续进化
- **Java 为骨、Python 为脑** — 大厂级微服务架构，业务与 AI 清晰分离

```

   设备/工单/权限/告警              异常检测/RUL/根因分析

InfluxDB + PostgreSQL + Kafka       飞书 Bot / 多维表格智能体
```
