# 第六梯队：开源数据集

> 可用于 PHM 算法训练、仿真数据引擎参考、方案 Demo 模型验证。

---

## 文件清单

| 文件 | 数据内容 | 推荐用途 |
|------|---------|---------|
| [NASA-IMS轴承数据集.md](./NASA-IMS轴承数据集.md) | 轴承 run-to-failure 振动数据，6.1 GB | RUL 算法基准测试 |
| [Kaggle泵数字孪生数据集.md](./Kaggle泵数字孪生数据集.md) | 150 台泵退化仿真，38 万行 | 仿真数据引擎参考模板 |
| [Kaggle泵传感器数据.md](./Kaggle泵传感器数据.md) | 1 台泵 220k 分钟，53 传感器 | 罕见故障预警算法 |
| [PHM-Society竞赛数据.md](./PHM-Society竞赛数据.md) | 历年竞赛数据集索引 | 数据格式与故障模式参考 |
