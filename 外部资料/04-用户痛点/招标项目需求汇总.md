# 水泵智能运维相关招标项目汇总（2024-2025）

> 来源：中国招标投标公共服务平台、深圳水务招标采购平台、广州/青岛/烟台公共资源交易平台。

## 项目一：太园泵站机组更新（二期）— 主机组与在线监测采购

| 项目 | 内容 |
|------|------|
| **招标单位** | 广东粤港供水有限公司 |
| **时间** | 2024 年 3 月 |
| **地点** | 东莞市桥头镇 |
| **预算** | **1,277.6 万元** |
| **内容** | 2 台主机组更新 + 在线监测设备（80 万元） |
| **关键要求** | 投标人须为水泵制造商，近 10 年有轴流泵/混流泵业绩 |
| **交货期** | 中标后 110 天 |

## 项目三：深圳污水处理厂站设备智能预测性运维系统建设

| 项目 | 内容 |
|------|------|
| **采购单位** | 深圳水务集团运营管理中心 |
| **时间** | 2025 年 11 月 |
| **预算** | **39.65 万元** |
| **内容** | 罗芳水质净化厂、大小坑泵站、蔡屋围泵站共 22 台关键水下设备，含传感器、数据采集站、本地服务器和软件系统 |
| **服务期** | 2 年（730 天） |
| **关键需求** | 振动/温度传感器、数据采集传输、本地化服务器 |

## 项目四：泵组智维管理设备采购

| 项目 | 内容 |
|------|------|
| **采购单位** | 深圳水务集团生产技术部 |
| **时间** | 2025 年 11 月 |
| **预算** | **9.8 万元** |

## 项目五：青岛引调水工程泵站预测性维护模型及系统试点研发

| 项目 | 内容 |
|------|------|
| **采购单位** | 青岛水投黄水东调投资运营有限公司 |
| **内容** | 开发预测性维护系统，实现健康状态实时掌握、故障精准定位、减少人工巡检、制定精准维护计划 |
| **关键需求** | 设备健康状态实时掌握、故障精准定位、减少人工巡检、降低运营成本 |

## 趋势总结

| 维度 | 特征 |
|------|------|
| **采购模式** | 从单纯设备采购转向"设备+在线监测+智能预测性维护"一体化 |
| **技术热点** | 振动分析、AI 智能诊断、设备健康度评估、物联网远程监测 |
| **项目规模** | 大型泵站改造（千万级）到中小型智能运维试点（十万级）全覆盖 |
| **覆盖区域** | 广东（深圳、东莞）、山东（烟台、青岛）较为活跃 |
| **采购主体** | 以水务公司/供水公司为主，多为国企自有资金 |

## 引用建议

> "从 2024-2025 年多个水务招标项目看，行业对泵站预测性维护系统需求明确且迫切：覆盖从十万级试点（如深圳水务 39.65 万元智能预测性运维项目）到千万级改造（如广东粤港供水 1,277 万元泵站机组在线监测项目）的全规模段，核心需求集中在振动分析、AI 智能诊断、设备健康度评估三个方向。"
